# Hull Tactical Market Prediction

Machine learning project for predicting market forward excess returns using XGBoost.

## Project Structure

- `train.ipynb` - Main training notebook with XGBoost model
- `hull-starter-notebook.ipynb` - Starter notebook
- `submission.csv` - Model predictions for submission
- `hull-tactical-market-prediction/` - Data directory
  - `train.csv` - Training dataset
  - `test.csv` - Test dataset
  - `kaggle_evaluation/` - Evaluation utilities

## Model

- **Algorithm**: XGBoost Regressor
- **Features**: Multiple feature groups (D, E, I, M, P, S, V prefixes)
- **Target**: Market forward excess returns
- **Validation**: 80/20 train-validation split

## Setup

1. Install required packages:
```bash
pip install pandas numpy xgboost scikit-learn matplotlib
```

2. Run the training notebook:
```bash
jupyter notebook train.ipynb
```
